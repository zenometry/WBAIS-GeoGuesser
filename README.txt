Put your image files in this folder:

  map.jpeg                 -> the campus map
  basketball-court.jpeg    -> a 360° panorama
  under-the-library.jpeg   -> a 360° panorama
  art-exhibit.jpeg         -> a 360° panorama
  ...                      -> add more as you add locations

The file names here must match the "pano" paths in js/locations.js
(and the map src in index.html).

Currently your photos live on Imgur. Download each one and save it
here with the matching name so the game no longer depends on Imgur.
